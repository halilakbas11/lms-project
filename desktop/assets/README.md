# LMS Desktop Icons

This folder should contain the application icons:

- `icon.ico` - Windows icon (256x256)
- `icon.icns` - macOS icon (512x512)
- `icon.png` - Linux/generic icon (512x512)

## Creating Icons

You can create icons using:
1. Online tools like https://icoconvert.com/
2. Image editing software like GIMP or Photoshop
3. Command-line tools like ImageMagick

## Requirements

- Windows: ICO format with 256x256 resolution
- macOS: ICNS format with 512x512 resolution
- Linux: PNG format with 512x512 resolution
